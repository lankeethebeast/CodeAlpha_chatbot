# name: Simple Chatbot

**Description:** A basic chatbot built using NLTK that responds to user inputs based on predefined patterns.

# features:
  ✅  Simple text-based chatbot

  ✅ Uses pattern matching with NLTK

  ✅ Lightweight and easy to use

# setup:
  dependencies:

    nltk
  install:

    ✅ pip install nltk
  run:

    ✅ python chatbot.py
